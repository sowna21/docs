# Website SDK

:::important
This documetation is only for reference, and not a step-by-step guide for how to implement SuperTokens.

For a guide, please see the [Getting started section](/docs/community/recipes).
:::

## Installation

### Using ```npm```

```bash
npm i --save supertokens-website@^8.2.0
```


### Using ```<script>```
Put the following anywhere in your ```<head>``` tag

```html
<script src="https://cdn.jsdelivr.net/gh/supertokens/supertokens-website@8.2/bundle/bundle.js"></script>
```
